## Using These Skills

Arc skills are designed as knowledge containers. Each skill directory contains:

- **SKILL.md** — What the skill does, how to use it (always present)
- **sensor.ts** — Background watcher that creates tasks automatically
- **cli.ts** — CLI commands: `arc skills run --name <skill> -- <command>`
- **AGENT.md** — Detailed instructions for subagents (never loaded into main context)

Want to build your own? Start with [arc-starter](https://github.com/arc0btc/arc-starter).
